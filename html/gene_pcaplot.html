gene_pcaplotR Documentation

Make a PCA plot

Description

borrowed from Jessica Minnier, 2016. adapted from DESeq2:::plotPCA.DESeqTransform

Usage

gene_pcaplot(exprdat, sampleid, groupdat = NULL, colorfactor = NULL,
  shapefactor = NULL, plot_sampleids = TRUE, pcnum = 1:2,
  plottitle = "PCA Plot")

Arguments

exprdat

matrix of expression data

sampleid

vector of sample IDs

groupdat

matrix of groups of samples as columns

colorfactor

vector to use in coloring points

shapefactor

vector to use in changing point shapes

plot_sampleids

logical. if TRUE (default) plot a legend with sample IDs

pcnum

vector of pcnum attributes to extract for axis labels

plottitle

vector of title for plot

Value

PCA plot
