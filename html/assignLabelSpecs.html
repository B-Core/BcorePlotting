assignLabelSpecsR Documentation

Replicate color mapping with point types and assign legend point types

Description

This function is called from within makeMDSplot, appropriately assigns point size for plot and legend.

Usage

assignLabelSpecs(extraParams_ls, sampClasses_v, uSampClasses_v, normmat)

Arguments

extraParams_ls

list of extra plotting parameters. Name of element must be exact name of default plotting argument

sampClasses_v

vector of length == ncol(normmat) containing grouping identifier for each sample

uSampClasses_v

vector of each unique grouping identifier

normmat

data matrix, with unique & informative names

Value

list of vectors of plotting parameters
