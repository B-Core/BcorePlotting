assignMappingSpecsR Documentation

Map colors to plotting factor and samples

Description

This function is called from within makeMDSplot, appropriately colors vectors and changes point size based on input

Usage

assignMappingSpecs(attribs, oneclass, colorspec, varPoints_v = NULL)

Arguments

attribs

list of sample classifications to be tracked in clustering. Each list element contains a string vector with one lable per sample.

oneclass

string name of attribs element to be used in MDS plot

colorspec

vector of color specifiers for colorRampPalette. colors are generated per sample by their clustering variable values

varPoints_v

vector of values to scale points by. Default is no scaling

Value

list of vectors of plotting paramters
